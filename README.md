## Installing
1. Clone this repository

```
git clone https://github.com/sabertooth9/chameleon-clustering.git
```

2. install package

```
pip install -e chameleon-clustering-master
```

3. Run sample code

```
cd chameleon-clustering-master
python -i example.py
```
